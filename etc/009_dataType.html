변수에 저장되는 데이터들의 타입을 알아보도록 하겠습니다.
  • 원시 데이터 타입(Primitive) - number, string, boolean, undefined, null, symbol
  • 객체 데이터 타입(Object) - function, object, array
      • symbol과 object는 함수를 먼저 살펴보아야 함으로 먼저 원시 데이터 타입을 살펴보도록 하겠씁니다.
      • number 데이터는 정수, 소수점, 지수를 표현할 수 있습니다.
      • string 데이터는 "",''로 표현합니다.
      • 문자 안에 문자를 표현할 수도 있습니다. "문자 안에 '문자'"
      • 특수한 목적을 위해 이스케이프 문자를 사용할 수도 있습니다.
